filter() 
      조건에 만족하는 것으로 새로운 배열을 반환한다
   

      구문
      arr.filter(callback(element[, index[, array]])[, thisArg])
      
      arr.filter(함수(현재값) {
        return 조건
      })

      arr.filter( (현재값) {
        return 조건
      })

      //결과 값이 1개면 아래와 같이 쓸 수 있다.
      arr.filter( (현재값) => 조건)



***반환값은 배열이다.



  매개변수

  callback:
   함수. 
   true를 반환하면 요소를 유지하고, 
   false를 반환하면 버립니다.
   (조건에 맞는 것만 반환하겠다 라는 뜻)
  
  다음 세 가지 매개변수를 받습니다.
  
  element:
  현재 요소.
  
  반환 값
  테스트를 통과한 요소로 이루어진 새로운 배열.
